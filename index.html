// FILE 01 — 我是誰

一名走遍走廊的老住民

後室不是地方，是一種狀態——當現實的縫隙裂開，你便落入這座無限延伸的舊辦公空間。 我研究後室多年，整理層級、記錄實體、收集每一份目擊報告。這座站，就是我的田野筆記。

// LV-0

黃室 The Lobby

最經典的入口層級。發黃壁紙、潮濕地毯、永不熄滅卻不斷閃爍的日光燈。安全，但會慢慢侵蝕你的理智。

相對安全無實體
// ITEM

杏仁水 Almond Water

後室中最珍貴的補給。能中和你體內逐漸「後室化」的成分，是每位住民的生命線。

必備補給
// RULE

三條鐵則

① 注意身後的聲音。② 別在同一處停留太久。③ 若聽見有人喊你的名字——別回應。

牢記在心

// ARCHIVE — 層級圖鑑

層級圖鑑 Levels

後室由無數「層級」組成，每一層都有自己的物理法則、危險度與居民。 以下是我親自踏查、整理出的經典層級檔案。危險條越紅，越別久留。

// LEVEL 0

黃室 The Lobby

無盡的單色辦公空間，潮濕地毯與發黃壁紙。日光燈持續嗡鳴。新住民最常落腳之地。

危險度 1/5有杏仁水
// LEVEL 1

習慣之地 Habitable Zone

昏暗的倉庫與停車場結構，偶爾有實體徘徊。比黃室更大、更陰冷，可作為臨時據點。

危險度 2/5少量實體
// LEVEL 2

面向後方 Pipe Dreams

狹長的維修隧道，牆面布滿滾燙管線。空間封閉、易迷路，且常有東西在管線後窺視。

危險度 3/5高溫
// LEVEL !

奔跑禁區 Run For Your Life

外觀類似 Level 1，但燈光轉紅時——奔跑。停下就是死亡。極度危險的高警戒層級。

危險度 5/5致命
// LEVEL FUN =)

歡樂層 Level Fun

永不結束的生日派對。糖果色牆面、氣球與蛋糕，看似歡樂，卻被「派對客」嚴密控制。微笑是強制的。

危險度 4/5派對客
// LEVEL 11

無盡都市 The Endless City

一座永恆白晝的城市，有道路、商店與長椅。相對安全，是住民社群最常聚集之處。

危險度 2/5有聚落
// LEVEL 37

汪洋之間 The Poolrooms

明亮、潔淨、充滿溫水的泳池迷宮。寧靜得近乎神聖，是後室中少見的療癒之地。

危險度 1/5療癒
// THE END

盡頭 The End

傳說中的最終層級。穿過那扇門，據說就能回到現實——但沒有人能證實，因為回去的人，再也沒回來說過。

未知傳說出口
檔案管理員筆記： 層級編號並不代表危險度高低，也不代表抵達順序。後室的拓樸是非歐幾里得的—— 你可能在黃室走一道門，就直接掉進 Level !。永遠帶著杏仁水。

// ARCHIVE — 實體檔案

實體檔案 Entities

後室裡你並不孤單。這些「實體」與你共享走廊——有的無害，有的會模仿人聲， 有的只要看你一眼，你就完了。以下是我整理的危險生物名錄。

// 實體 0

笨拙 Clumps

由毛髮、皮膚與牙齒纏成的肉團，緩慢地在地上滾動。行動遲緩，但別讓它碰到你——它會把你也捲進去。

敵對移動緩慢
// 實體 1

獵犬 Hounds

瘦長、人形、長著利齒的掠食者。視力差，但聽覺敏銳。保持安靜，緩慢移動，別讓它聽見你的腳步。

高度敵對聽聲辨位
// 實體 3

傷痕史密斯 Smilers

黑暗中漂浮的一張嘴與兩隻眼。只在低光環境出現。看見它時別直視，緩緩退向有光處。

敵對畏光
// 實體 7

那個女人 The Woman

遠看是溫柔的女子，會喊出你熟悉之人的聲音引你靠近。靠近後——別。它已奪走無數想家的住民。

致命擬聲
// 派對客

派對客 Partygoers

Level Fun 的居民。戴著詭異面具、強迫你「享受派對」的人形實體。不微笑、想離開的人會被它們「處理」。

敵對群體行動
// 友善實體

史密斯先生 Mr. Smiles

少數友善的實體之一。會與住民交易杏仁水與物資。對它有禮貌，它便是黑暗中難得的盟友。

友善可交易
// 實體 96

面容竊賊 Skin-Stealers

能模仿人類外型，混入住民群體。辨識法：它的動作略顯僵硬，皮膚邊緣有不自然的接縫。多疑能保命。

致命擬態
// 守護者

看守者 The Watchers

並非攻擊性實體，只是「看著」。當你感到背後有視線卻空無一人時，它就在那。無害，但會慢慢磨耗你的神智。

中立精神壓迫
生存提示： 大多數實體都怕光與聲響的突變。隨身攜帶光源，保持冷靜， 記住——恐慌比實體更致命。

// ABOUT — 關於後室

什麼是後室？

Backrooms 是一個源自網路的都市傳說與恐怖創作宇宙。它把「既視感」化作一座無限的房間。

// ORIGIN

起源於一張照片

2019 年，一張模糊的發黃空房間照片配上一段文字出現在網路論壇，描述「不小心 no-clip 出現實」會掉進的地方。一張圖，點燃了整個宇宙。

// NO-CLIP

什麼是「無剪輯」

「No-clip」借自電玩術語，指角色穿過了本不該穿過的牆。在後室世界觀裡，現實偶爾會出現破綻，讓人「剪輯」出去，墜入後室。

// VIBE

閾限空間美學

後室的核心是 Liminal Space——那些熟悉卻空無一人的過渡空間：深夜的走廊、放假的學校、無人的泳池。既懷念，又毛骨悚然。

// LORE

由眾人共筑的世界

後室沒有單一作者。成千上萬的粉絲共同撰寫層級、實體與規則，讓它成長為一個龐大的協作虛構宇宙——Backrooms Wikidot 收錄了上千個層級。

// MEDIA

從迷因到電影

創作者 Kane Pixels 的《The Backrooms (Found Footage)》系列影片爆紅，讓後室從文字迷因躍上影像，甚至傳出好萊塢改編電影的計畫。

// CORE

三個關鍵字

發黃 (yellow)、嗡鳴 (hum)、無盡 (endless)。只要掌握這三點，你就懂了後室的靈魂。其餘的恐怖，留給你自己去走廊裡發現。

免責聲明： 後室是虛構的網路創作 (creepypasta / collaborative fiction)。本站為粉絲自製的致敬作品， 不代表任何官方，也請放心——你並不會真的掉進去……大概吧。

// MANUAL — 生存指南

生存指南 Survival

掉進後室不是世界末日——只要你夠冷靜、夠謹慎。這是我多年踏查總結出的保命守則， 照著做，你活下去的機率會高很多。

第一補給：杏仁水

後室中最重要的資源。它能補水、補充能量，並延緩你的身體「後室化」。 看見杏仁水，務必收下——它可能就是你回家的關鍵。

// RULE 01

保持冷靜

恐慌會讓你迷路、喘息、發出聲響，引來實體。深呼吸，告訴自己：這只是一個很大的房間。

// RULE 02

持續移動，但別狂奔

停留太久會被「看守者」鎖定；狂奔則會耗盡體力又製造噪音。穩定的步伐才是王道。

// RULE 03

標記你的路徑

後室的走廊會自我重複甚至改變。用粉筆、撕下的紙片標記轉角，避免在同一片黃牆裡繞圈。

// RULE 04

聽見名字別回應

那個女人、擬聲實體會用你親友的聲音呼喚你。在後室裡，任何呼喊你名字的聲音都該被懷疑。

// RULE 05

找到「同盟」

後室裡有住民社群 (如 M.E.G. 探險隊)。團體比孤狼安全。但也提防「面容竊賊」混入——保持警覺。

// RULE 06

記錄一切

畫地圖、寫日誌、記下每個門通往哪。你的筆記不只救自己，也是留給下一位迷途者的禮物。

關於「離開」： 沒有人能保證一定回得去。傳說 The End 是出口，也有人靠著特定層級的後門返回現實。 最實際的建議是：先活下去，再談離開。
給新住民的一句話： 你不是第一個掉進來的，也不會是最後一個。這片走廊很可怕，但你並不孤單。撐住。
