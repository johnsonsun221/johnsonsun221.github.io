// 03 — ABOUT

什麼是後室？

Backrooms 是一個源自網路的都市傳說與恐怖創作宇宙。它把「既視感」化作一座無限的房間——熟悉、空曠、令人不安。

ORIGIN2019

起源於一張照片

一張模糊的發黃空房間照片配上一段文字出現在網路論壇，描述「不小心 no-clip 出現實」會掉進的地方。一張圖，點燃了整個宇宙。

NO-CLIP↯

什麼是「無剪輯」

「No-clip」借自電玩術語，指角色穿過了本不該穿過的牆。在後室世界觀裡，現實偶爾會出現破綻，讓人「剪輯」出去，墜入後室。

VIBE◷

閾限空間美學

後室的核心是 Liminal Space——那些熟悉卻空無一人的過渡空間：深夜的走廊、放假的學校、無人的泳池。既懷念，又毛骨悚然。

LORE∞

由眾人共筑的世界

後室沒有單一作者。成千上萬的粉絲共同撰寫層級、實體與規則，讓它成長為一個龐大的協作虛構宇宙——Wikidot 收錄了上千個層級。

MEDIA▶

從迷因到電影

創作者 Kane Pixels 的《The Backrooms (Found Footage)》系列影片爆紅，讓後室從文字迷因躍上影像，甚至傳出好萊塢改編電影的計畫。

CORE3

三個關鍵字

發黃 (yellow)、嗡鳴 (hum)、無盡 (endless)。只要掌握這三點，你就懂了後室的靈魂。其餘的恐怖，留給你自己去走廊裡發現。

// ANALYSIS — 深度分析

為什麼我們害怕空房間 Deep Dive

身為資深粉絲，我不只收集設定——我想搞懂它為什麼有效。以下是我多年觀察後室現象整理出的五個分析切角。

ANALYSIS 01心理

凱諾普西亞 Kenopsia

「本該擠滿人的地方，空無一人」的詭異氛圍，有個專門的造詞：kenopsia。商場、教室、辦公室在我們的記憶裡永遠伴隨人聲；把人抽走，大腦的預測模型就會失靈——熟悉感瞬間翻轉成警報。後室做的事，是把這種失靈拉到無限大：六億平方公里的「應該有人」。

ANALYSIS 02敘事

恐怖的低成本革命

傳統恐怖片靠怪物特寫與 jump scare 堆疊刺激；後室幾乎反著做——最可怕的時刻，畫面裡什麼都沒有，只有黃牆與嗡鳴。實體出場率被刻意壓到極低，恐怖的主體從「看見什麼」轉移到「預期會看見什麼」。這是恐怖敘事的範式轉移：氛圍即怪物。

ANALYSIS 03認知

非歐拓樸與認知地圖

人類導航依賴「空間一致性」：走過的走廊應該還在原處。後室的走廊會重複、會改變、門不守恆——這正是夢境與迷路噩夢的底層機制。所以每個人第一次看到後室都「似曾相識」：你不是沒去過，你在夢裡迷路過。

ANALYSIS 04傳播

模因式協作創作

後室沒有版權方、沒有正典委員會。一張圖 → 一段貼文 → 上千個層級。它的 wiki 借用了 SCP 基金會的「偽官方文件體」：用冷靜的分級、編號、生存須知去描述瘋狂的內容——格式本身就是恐怖的放大器。可信的口吻，比血腥的描寫更讓人發毛。

ANALYSIS 05影像

Kane Pixels 的可信度工程

《The Backrooms (Found Footage)》用 1990s VHS 質感包裹現代 3D 動畫：低解析度與雜訊一邊遮蔽 CG 的破綻，一邊觸發「舊錄影帶＝真實檔案」的條件反射。手持鏡頭的喘息節奏取代配樂——這是一套精準的可信度工程，也是它能假亂真爆紅的原因。

CONCLUSION∎

我的結論

後室之所以迷人，是因為它是一面「集體寫成的鏡子」：我們真正害怕的不是怪物，而是現代生活本身的空轉——無盡的辦公室、重複的走廊、永遠亮著卻照不暖人的燈。每個寫層級的人，都在描述自己的某一天。

// MANUAL — 生存指南

生存指南 Survival

掉進後室不是世界末日——只要你夠冷靜、夠謹慎。這是我多年踏查總結出的保命守則，照著做，你活下去的機率會高很多。

// SUPPLY 01

第一補給：杏仁水

後室中最重要的資源。它能補水、補充能量，並延緩你的身體「後室化」。看見杏仁水，務必收下——它可能就是你回家的關鍵。

RULE 0101

保持冷靜

恐慌會讓你迷路、喘息、發出聲響，引來實體。深呼吸，告訴自己：這只是一個很大的房間。

RULE 0202

持續移動，但別狂奔

停留太久會被「看守者」鎖定；狂奔則會耗盡體力又製造噪音。穩定的步伐才是王道。

RULE 0303

標記你的路徑

後室的走廊會自我重複甚至改變。用粉筆、撕下的紙片標記轉角，避免在同一片黃牆裡繞圈。

RULE 0404

聽見名字別回應

那個女人、擬聲實體會用你親友的聲音呼喚你。在後室裡，任何呼喊你名字的聲音都該被懷疑。

RULE 0505

找到「同盟」

後室裡有住民社群 (如 M.E.G. 探險隊)。團體比孤狼安全。但也提防「面容竊賊」混入——保持警覺。

RULE 0606

記錄一切

畫地圖、寫日誌、記下每個門通往哪。你的筆記不只救自己，也是留給下一位迷途者的禮物。

關於「離開」： 沒有人能保證一定回得去。傳說 The End 是出口，也有人靠著特定層級的後門返回現實。最實際的建議是：先活下去，再談離開。
給新住民的一句話： 你不是第一個掉進來的，也不會是最後一個。這片走廊很可怕，但你並不孤單。撐住。
